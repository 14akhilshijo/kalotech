## üé® **Customization Options**

### **Colors**
Update `tailwind.config.js` to modify the color scheme:
```javascript
colors: {
  'deep-blue': '#0F172A',
  'teal': '#0D9488',
  // ... more colors
}
```

### **Animations**
Customize animations in `tailwind.config.js`:
```javascript

  'float': 'float 8s ease-in-out infinite',
  'morph': 'morph 8s ease-in-out infinite',
  // ... more animations
}
```
